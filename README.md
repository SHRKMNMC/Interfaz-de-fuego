# Interfaz-de-fuego

Interfaz que simula un fuego, el cual se enciende mediante unas chispas en su base que propaga temperaturas hacía arriba. Estas tienen asociados los colores de la llama de acuerdo a sus temperaturas.

Las temperaruras se propagan hacia arriba y ambnos lados, haciendo crecer la llama, sin embargo, también hay un enfriamiento.
